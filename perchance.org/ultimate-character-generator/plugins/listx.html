Select ListX Plugin

Random selection utilities for nested list structures

1. selectListX - Select ONE leaf

Selects a single random leaf from nested structures.

2. selectListsX - Select MANY leaves

Selects multiple random leaves. Supports: single number, [min,max] array, or multiple numbers to pick from.

3. selectAllListsX - Get ALL leaves

Recursively gathers all leaf values from nested structures.
