📋 listX Plugin — Ultimate Character List 📋

Import in Perchance code panel:

listX = \{import:listx-plugin\}

This plugin provides a comprehensive, hierarchical, and extensible set of lists for character generation, covering identity, appearance, personality, background, skills, relationships, adult content, and more.

  • All standard and extended character attributes
  • Supports leaf selection, random sampling, and unique draws
  • Ready for use with selectLeaf, selectLeaves, selectAllLeaves, and listX

Example usage:

output
  [character_first_name_male] [character_last_name_fantasy] is a [character_age_young_adult] [character_gender_female_pronouns] [character_species_elf] with [character_hair_color_silver] hair and [character_personality_trait_positive] personality.
  Random leaf: [selectLeaf(character_first_name_male)]
  5 unique leaves: [selectLeaves(character_first_name_male, 5).joinItems(", ")]
  5-10 random unique leaves: [selectLeaves(character, 5, 10).joinItems(", ")]

See listx-plugin.perchance for the full list structure and all available attributes.

Sections included:

  • Identity (name, age, gender, species, sexuality, pronouns, etc.)
  • Appearance (skin, eyes, hair, height, build, genitals, modifications, etc.)
  • Personality (traits, quirks, alignment, values, etc.)
  • Background (origin, culture, occupation, education, wealth, etc.)
  • Skills & Abilities (combat, magic, talents, languages, etc.)
  • Relationships (family, friends, rivals, romance, etc.)
  • Equipment (clothing, weapons, items, etc.)
  • Adult Content (kinks, fetishes, acts, power dynamics, etc.)
  • Portrait (style, pose, clothing, accessories, lighting, etc.)
  • Custom/Extensible fields

For more info, see perchance.org/plugins.
