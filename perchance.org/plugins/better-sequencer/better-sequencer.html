Item: [randomList()]

Items in a list will be selected in order until all items are selected.

To use it:

\[betterSEQUENCE(",", "Name", "word1, word2, word3")\]
  • The first "," is the character used to separate the items. If your list has items separated by commas (for example, "item1, item2, item3"), use a comma. If they are separated by * (for example, "item1 * item2 * item3"), use "*", and so on.
  • "Name" is the name of the list that will be created only when the function is used. You can use any name; it is only used to make the variable unique. Do not use spaces, and avoid names starting with numbers or special characters. Just use a short name — it does not need to be meaningful.
  • The last parameter can be your list written inside quotation marks, using the chosen separator (as shown in the example). In a special case, you can also pass an existing variable or Perchance list (without quotation marks), like this:
    \[betterSEQUENCE(",", "list472", anotherRandomList)\]
  • For items separated by new lines or line breaks, use "\\n".

To import into your generator (for builders or generator owners):

betterSEQUENCE = \{import:better-sequencer\}
