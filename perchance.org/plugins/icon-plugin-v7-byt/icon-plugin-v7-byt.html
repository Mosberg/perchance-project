Font Awesome v7 Icon Plugin

This is a forked version of the original icon plugin v6 and icon plugin that uses the new Font Awesome 6.7 and 7 source that brings new icons and design changes. Note that some icon names from v6 and olver versions might be different from the old one, so you might need to re-arrange some code if you want to upgrade to this plugin.

All you need to do is use the full name of the icon, like 'fa-solid fa-icon-name'

If you want to use the older 6.6 or 5.8.1 icons, use the original ones instead, or change whatever is in the name to the format above (fa-solid fa-icon-name), check the awesome fonts site to see the icons names.

This plugin allows you to use icons from Font Awesome.

Put this in your Perchance code: icon = \{import:icon-plugin-v7-byt\}


Once you've done that, you can use icons by referencing their names: 

\[icon("magnifying-glass")\]
[$output("magnifying-glass")]

You can also add styles: 

\[icon("flag", "color: red;")\]
[$output("flag", "color: red")]
\[icon("bolt", "background-color: #ffcc00; color: white; padding: 0.2em; font-size: 19px; border-radius: 3px;")\]
[$output("bolt", "background-color: #ffcc00; color: white; padding: 0.2em; font-size: 19px; border-radius: 3px;")]

To shorten your code, use a class or an id: 

<style>
  .energy \{
     background-color: #ffcf10;
     color: white;
     padding: 0.2em;
     font-size: 19px;
     border-radius: 3px;
  \}
</style>
\[icon("bolt", ".energy")\]
[$output("bolt", ".energy")]

This also allows you to use pseudo classes:
(Try hovering over the icon)

<style>
  .energy \{
    background-color: #ffcf10;
    color: white;
    padding: 0.2em;
    font-size: 19px;
    border-radius: 3px;
    transition: color 0.4s, background-color 0.4s;
  \}
 .energy:hover \{
    color: #ffcf10;
    background-color: white;
  \}
</style>
\[icon("bolt", ".energy")\]
[$output("bolt", ".energy-hover")]

Not mentioned in the original plugin:
You can also apply Font Awesome v6's built-in animations to the icon with classes, too:

\[icon("star", ".fa-beat")\]
[$output("star", ".fa-beat")]
\[icon("compact-disc", ".fa-flip")\]
[$output("compact-disc", ".fa-flip")]
\[icon("bomb", ".fa-shake")\]
[$output("bomb", ".fa-shake")]
\[icon("spinner", ".fa-spin")\]
[$output("spinner", ".fa-spin")]
\[icon("spinner", ".fa-spin fa-spin-reverse")\]
[$output("spinner", ".fa-spin fa-spin-reverse")]

You can even set their properties (so you could make them slower, faster, or behave differently):

<style>
  .reverseflip \{
    --fa-flip-x: 1;
    --fa-flip-y: 0;
  \}
</style>
\[icon("heart", ".fa-flip reverseflip")\]
[$output("heart", ".fa-flip reverse-flip")]
<style>
  .spinspeed \{
    --fa-animation-duration: 4s;
  \}
</style>
\[icon("spinner", ".fa-spin spinspeed")\]
[$output("spinner", ".fa-spin spinspeed")]

And combine them with pseudo-class styling:

<style>
  .heart-hover \{
     background-color: #ff3040;
     color: white; padding: 0.2em;
     border-radius: 3px;
     transition: color 0.4s, background-color 0.5s;
     --fa-beat-scale: 1.2;
  \}
  .heart-hover:hover \{
     color: #ff3040;
     background-color: white;
     --fa-animation-duration: 2.5s;
     --fa-beat-scale: 1.5;
  \}
  .reverseflip \{
     --fa-flip-x: 1;
     --fa-flip-y: 0;
  \}
</style>
\[icon("heart", ".fa-beat heart-hover")\]
[$output("heart", ".fa-beat heart-hover")]

These are just some examples on the animations, you can refer to this page for more supported animation classes.
