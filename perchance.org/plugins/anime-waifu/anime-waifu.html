Anime Waifu List

Work in progress...
This list contains only fictional female characters from anime and games. It does not include characters from films or other realistic works, unless an 'anime version' exists.
Please note that this list is nothing more than text, textual descriptions of characters; this is not a LoRA (Low-Rank Adaptation). If you want more options, different Art Styles, etc., use a generator that has this list, like Ai Art Generator-Advanced.
Help improve the list
You can create a character description and submit it to be added to the list. The description must follow the format below:

    Notes:
  • Before submitting, check if the generated character image is approximately 50% accurate by clicking 'Generate 💖'. Characters that don't have enough data for good images will not be added to the list. If the generated image doesn't match the original character at all, don't waste your time submitting it (not your fault, the model simply lacks data).
  • The AI cannot faithfully recreate most characters; this is normal.
  • Sometimes 'less is more,' for example, a character with 'parted bangs' performs better WITHOUT needing to use the 'parted bangs' tag, as the tag would force generic parted bangs results and distort the character.
  • Fill the clothes/outfit input.
  • I will manually check all submitted descriptions. As you can see, this is time-consuming. Good descriptions will be copied and pasted into the list, saving time and therefore making the process faster.
  • The generated character image needs to be consistent and similar, not just a poorly made imitation. If the description fails to maintain quality and consistency, it's because the model doesn't have enough data about the character, and in that case, it's recommended to give up and not waste time. If you own an 'Ai Perchance Generator', you can import this list with: AWL = \{import:anime-waifu\} Then, in the 'settings', inside 'userInputs', you can use something like (example):
animeCharacter   label = Waifu   type = select   options     meta:import       from = \[AWL\]
You also need to create a variable to enable/disable the characters' outfits (otherwise they will only have basic descriptions of their body, eyes, and hair). Simply place this in the left panel →:
<script>   window.characterOutfit = "active"; </script>

You can also create a button to turn the outfits on and off. Here's a basic example: <button onclick='window.characterOutfit = window.characterOutfit === "active" ? "inactive" : "active"'>toggle Outfit</button>
This list is like an Art Style; you simply call it with 'name.prompt', where 'name' is where you imported it, for example, artStyle3.prompt, or animeCharacter.prompt, etc. You can also use just the characters' outfits. Instead of using '.prompt', just call '.outfit'.
