AI Text Rewriter Framework v2

Advanced parser-safe UI framework for Perchance text rewriting generators.

Key Features

  • Multi-pack mode loading with search, pack filter, favorites, and recent mode chips.
  • Global prefix/suffix rewrite instructions and advanced rewrite controls.
  • Profiles for reusable configurations and local snapshot history management.
  • Single rewrite and batch rewrite workflows with progress/status hooks.
  • Output tools: copy, download, stats, and quick diff preview.

Expected Generator Hooks

  • generate() for single rewrite execution.
  • generateBatchRewrite() for batch processing.
  • generateModesHtml() and generateModesDataJs().
