AI Text Rewriter Framework

Reusable framework plugin for Perchance text-rewriter generators.

Usage pattern:

  • Import this plugin as generateHTML in your generator.
  • Provide generator-side settings and a generate() function.
  • Import mode packs (for example ai-text-rewriter-prompt-default) and assign to modes.
  • Render the panel by putting \[generateHTML(settings)\] in the generator HTML file.
