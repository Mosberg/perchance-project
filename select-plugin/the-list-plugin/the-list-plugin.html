The List Plugin

The List Plugin combines select/list utilities from all the select-plugin helpers into a single importable module. It exposes list selection, filtering, joining, consumable list behavior, range utilities, and lockable/locker state controls.

Import it in your lists editor like this:

listUtils = \{import:the-list-plugin\}

Once imported, you can use it directly as a weighted selector, or access its helper methods.

Examples

select = \{import:the-list-plugin\}

a =
  apple
  banana
  cherry

b =
  red
  blue
  green

x = \[select(a, b, null, 1, 2)\]

This chooses either a or b, with odds 1:2.

Helper methods include:

  • select.selectLeaf(list): choose a leaf node from a nested list
  • select.selectLeaves(list, count): choose multiple leaves
  • select.selectAllLeaves(list): flatten list to all leaves
  • select.selectRange(list, start, end): slice a list range
  • select.selectUntil(list, predicate): select until a condition matches
  • select.joinLists(a, b, ...): concatenate several lists
  • select.filterList(list, fn): keep items that pass a callback
  • select.excludeItems(list, item1, item2, ...): remove named items
  • select.consumableList(list): create a consumable list that replenishes when exhausted
  • select.consumableLeafList(list): consume leaves without repeating
  • select.lockable(list): lock list output with an inline toggle
  • select.locker(name, value): store a value across render passes
  • select.lockButton(name): render a locker toggle button

For example, to exclude unwanted terms:

remaining = \[select.excludeItems(myList, "skipMe", anotherItem)\]

To create a locked list output:

locked = \[select.lockable(myList)\]
button = \[select.lockButton("session")\]

This plugin is intended as a single combination point for all the select-plugin features in the select-plugin directory.


Notes:

  • Use select = \{import:the-list-plugin\} to index every helper from one import.
  • Direct function call select(...) uses the random select behavior.
  • See the method names above for list-specific helpers.
