The List Plugin v2

The List Plugin v2 exposes enhanced list utilities for nested Perchance lists and object trees. It supports ordering, filtering, randomization, weighted selection, path-based access, and more.

Import it into your lists editor like this:

listUtils = \{import:the-list-plugin-v2\}

Available functions

  • listUtils.selectOne(list)
  • listUtils.selectAll(list)
  • listUtils.selectMany(list, count)
  • listUtils.selectSpecific(list, index)
  • listUtils.selectOrder(list, 1, 3, 2)
  • listUtils.selectRange(list, start, end)
  • listUtils.selectExcept(list, 2, 4)
  • listUtils.selectUnique(list, count)
  • listUtils.selectShuffle(list)
  • listUtils.selectWeighted(list, weights)
  • listUtils.selectPath(root, "category.subcategory")
  • listUtils.selectWhere(list, item => item.includes("foo"))

Example

listUtils = \{import:the-list-plugin-v2\}

randomIdentity = [character.identity.selectOne]
allMaleNames = [character.identity.name.first.male.selectAll.joinItems(", ")]
selectManyFemaleNames = [character.identity.name.first.female.selectMany(3,5).joinItems(", ")]

The plugin is built to work with large nested list definitions like list.perchance. It works on both native Perchance list objects and nested object trees.


Notes:

  • selectAll returns a list-like array of candidate values.
  • selectMany accepts a fixed count or range like (1,3).
  • selectWhere accepts a callback predicate.

[character.identity.name.first.male.selectOne] [character.identity.name.first.male.selectMany(3).joinItems(", ")] is a [character.joinItems(", ")] Name: [character.identity.name.first] [character.identity.name.last.common] Age: [character.identity.age] Gender: [character.identity.gender] Species: [character.identity.species] Pronouns: [character.identity.pronouns] Species: [character.identity.species.selectOne]
